# Fase 3: Pipeline de Dataflow

## Objetivo

Procesar la telemetría de drones en tiempo real usando Apache Beam en Dataflow.

## Componentes clave

- Lectura de Pub/Sub
- Transformaciones y validaciones
- Windowing y agregaciones
- Escritura a BigQuery

## Pasos detallados

1. Configurar pipeline de Beam
2. Implementar transformaciones y validaciones
3. Aplicar windowing (FixedWindow 60s)
4. Agregar por drone_id
5. Escribir a BigQuery
