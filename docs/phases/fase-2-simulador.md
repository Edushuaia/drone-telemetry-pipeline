# Fase 2: Simulador de Drones

## Objetivo

Desarrollar un simulador que genere telemetría realista de múltiples drones y la publique en Pub/Sub.

## Componentes clave

- Generador de trayectorias GPS
- Simulación de sensores (batería, velocidad, temperatura, etc.)
- Publicador de mensajes a Pub/Sub

## Pasos detallados

1. Definir el esquema de telemetría
2. Implementar el generador de trayectorias
3. Simular sensores y eventos
4. Publicar mensajes a Pub/Sub
5. Añadir tests unitarios

## Buenas prácticas

- Modulariza el código del simulador
- Usa variables de entorno para configuración
- Añade logs para depuración
